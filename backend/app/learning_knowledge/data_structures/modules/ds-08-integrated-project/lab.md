# 实验：课程项目 — 数据结构可视化学习工具

## 任务目标
综合运用本课程所学数据结构，实现一个小型可视化学习工具。

## 输入输出
- 输入：用户从以下选题中选择其一（或自拟主题经批准）
  1. 排序算法可视化器：实时展示冒泡、插入、快排、归并的执行过程
  2. 迷宫生成与求解器：用 DFS/BFS 生成迷宫并寻找出口
  3. 简易文件系统模拟：树形目录结构 + 文件增删查改
  4. 贪吃蛇（数据结构版）：deque 蛇身 + set 碰撞检测 + 排行榜
  5. 表达式求值与可视化：中缀→后缀→语法树→计算
- 输出：可运行的项目源码 + 设计文档 + 运行截图

## 实现步骤
1. 数据结构类、算法函数、UI/展示层分离设计
2. 至少使用 3 种不同的数据结构
3. 编写基础测试用例
4. 完成设计文档（≤500 字）：问题描述、数据结构选型理由、复杂度分析
5. 编写 README 说明如何启动

## 提交物
1. 项目源码（含 README.md）
2. 设计文档（`design.md`，≤500 字）
3. 运行截图或 GIF（展示核心功能）
4. 简要自评：选型是否合理？复杂度是否可接受？可改进之处？

## 反思问题
1. 项目中最大的选型挑战是什么？你最终如何决策？
2. 如果项目规模扩大 10 倍，哪些数据结构选择需要重新考虑？
3. 回顾整个课程，你对数据结构的理解从"死记硬背"转变为"场景匹配"了吗？
