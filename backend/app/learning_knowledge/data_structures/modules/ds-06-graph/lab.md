# 实验：城市路线规划器

## 任务目标
使用图算法构建一个简单的城市路线规划器，支持最短路径和最小生成树计算。

## 输入输出
- 输入：城市地图（节点 = 城市，边 = 道路，权 = 距离或时间）
- 输出：两城市间的最短路径、连接所有城市的最少总里程

## 实现步骤
1. 定义至少包含 8 个城市和 12 条道路的地图数据
2. 使用邻接表存储图
3. 实现以下功能：
   - `shortest_path(start, end)`：Dijkstra 算法，返回路径和总距离
   - `mst()`：Prim 或 Kruskal 算法，返回最小生成树的边集合
   - `bfs_distance(start)`：返回 start 到各城市的跳数（忽略距离）
4. 编写简单的命令行交互界面或测试函数展示结果
5. 设计至少 3 个测试场景（如：最短路径、绕路情况、MST 对比）

## 提交物
1. `city_router.py` — 图结构和算法实现
2. 测试输出截图
3. 数据集说明（城市地图的图示或表格）
