# 实验：表达式求值器

## 任务目标
实现一个能处理四则运算和括号的表达式求值器，理解栈在编译原理中的应用。

## 输入输出
- 输入：中缀表达式字符串（含 +、-、*、/、括号、非负整数）
- 输出：计算结果（浮点数）

## 实现步骤
1. 实现 `infix_to_postfix(expr)` 函数：
   - 支持运算符 +、-、*、/
   - 正确处理运算符优先级（* / 高于 + -）
   - 正确处理括号
   - 处理多位数字和空格
2. 实现 `evaluate_postfix(postfix)` 函数：
   - 用栈计算后缀表达式
   - 处理除零异常
3. 实现 `evaluate(expr)` 一站式求值
4. 编写测试用例覆盖：
   - `"3 + 4 * 2"` → 11
   - `"(1 + 2) * (3 + 4)"` → 21
   - `"10 / 2 + 3"` → 8.0
   - `"1 / 0"` → 报错

## 提交物
1. `expression_evaluator.py` — 完整求值器
2. 包含至少 8 个测试用例的测试函数
3. 流程图说明中缀转后缀的步骤
