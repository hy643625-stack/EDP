# 实验：二叉树遍历与堆排序

## 任务目标
实现二叉树的构建、遍历和可视化，以及堆排序的完整实现。

## 输入输出
- 输入：一组整数序列
- 输出：二叉树的四种遍历结果 + 堆排序后的有序序列 + 堆的构建过程

## 实现步骤
1. 实现 `TreeNode` 类和 `BinaryTree` 类
   - `from_list(values)`：从层序列表构建二叉树
   - 递归和非递归两种方式实现前序、中序、后序遍历
   - 层序遍历（队列实现）
2. 实现堆操作：
   - `build_heap(arr)`：Floyd O(n) 建堆
   - `heap_sort(arr)`：堆排序
3. 实现一个简单的树形打印函数：
   ```
       15
      /  \
     8    20
    / \   / \
   6  12 18 25
   ```
4. 使用 `heapq` 实现找到列表中第 k 大的元素
5. 编写测试验证所有功能

## 提交物
1. `binary_tree.py` — 二叉树实现
2. `heap_operations.py` — 堆操作实现
3. 控制台运行的演示输出截图

## 反思问题
1. 递归遍历和非递归遍历在空间复杂度上有何差异？
2. 建堆的 O(n) 算法为什么比 n 次插入 O(n log n) 更优？
3. 堆排序在什么场景下比归并排序更有优势？
