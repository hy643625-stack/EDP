# 树与二叉树

## 学习目标
- 能区分树的常用术语（根、叶、深度、高度、度）
- 能实现二叉树的四种遍历（前序、中序、后序、层序）
- 能解释堆的性质及其在优先队列中的应用
- 能构造哈夫曼树并计算带权路径长度 WPL

## 知识地图
```
树结构
├── 基本概念（根、叶、深度、高度、度、层）
├── 二叉树
│   ├── 遍历（前序/中序/后序/层序）
│   ├── 二叉搜索树 BST
│   │   ├── 查找 O(log n) ~ O(n)
│   │   ├── 插入 / 删除
│   │   └── 平衡问题（AVL、红黑树）

```

## 核心讲解

### 树的基本术语
- **节点的度**：该节点的子树个数
- **树的度**：所有节点度的最大值
- **叶节点**：度为 0 的节点
- **深度**：从根到该节点的边数（根深度为 0）
- **高度**：从该节点到最深叶的边数

### 二叉树遍历
| 遍历方式 | 顺序 | 非递归实现 | 典型用途 |
|---------|------|-----------|---------|
| 前序 | 根→左→右 | 栈 | 复制树、前缀表达式 |
| 中序 | 左→根→右 | 栈 | BST 输出有序序列 |
| 后序 | 左→右→根 | 双栈/标记法 | 删除树、后缀表达式 |
| 层序 | 逐层 | 队列 | BFS、求树的宽度 |

### 堆
堆是用数组存储的完全二叉树。核心操作：
- **插入**：将新元素放在末尾，然后"上浮"到正确位置——O(log n)
- **删除极值**：将末尾元素移到根，然后"下沉"到正确位置——O(log n)
- **建堆**：Floyd 算法从最后一个非叶节点向前 heapify——O(n)

Python 的 `heapq` 模块提供的是小顶堆。

### 哈夫曼树
构造步骤：
1. 将所有节点按权值放入优先队列（小顶堆）
2. 取出两个最小节点，构造新节点（权值 = 两子之和）
3. 新节点放回队列
4. 重复直到只剩一个节点

## 典型例子
1. **验证 BST**：递归传递上下界
2. **Top-K 问题**：维护大小为 k 的小顶堆
3. **堆排序**：建堆 O(n) + n 次删除 O(n log n)
4. **哈夫曼编码**：文件压缩的经典算法

## 易错点
- BST 验证时只比较直接父子 → 应该传递区间 [min, max]
- 堆不是完全排序的，只有根元素是最值
- 层序遍历用栈而非队列 → 结果错误
- 哈夫曼树构造时使用大顶堆 → 应使用小顶堆

## 与后续模块的关系
树结构是图和搜索算法的基础。BST 的平衡问题引出后续的 AVL 和红黑树。堆是 Dijkstra 最短路径和 Prim 最小生成树算法的核心数据结构。
