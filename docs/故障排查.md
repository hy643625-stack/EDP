# 故障排查

## 1. 启动后前端显示“无法连接服务”
- 确认后端是否在运行：`http://127.0.0.1:18765/health`
- 若无响应，重新执行：`.\start_everydayperfect.ps1`
- 检查端口占用：`18765` 和 `5173`

## 2. 桌面版出现请求超时
- 先重启应用。
- 若仍异常，查看日志：`%LOCALAPPDATA%\EveryDayPerfect\logs\desktop-launcher.log`
- 优先使用最新安装包版本。

## 3. 任务创建无反应
- 打开浏览器开发者工具查看接口请求是否失败。
- 后端接口 `/v1/tasks` 返回 `success: true` 但页面不更新时，刷新页面并重试。

## 4. 脚本执行被禁止
在 PowerShell 执行：
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

## 5. 仍无法定位
- 截图报错页。
- 复制诊断信息（应用内按钮）。
- 一并提供日志文件。
